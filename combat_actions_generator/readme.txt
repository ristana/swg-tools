#Purpose
This script is for easy generation of entries into combat_actions.java

#Requirements
Python3

##How To Use
ensure in.txt is present in root directory
ensure out.txt is present in root directory

input a list of desired entries to be generated (single line breaks) into in.txt

run the script
>>> E:\GitHub\swg-tools\combat_actions_generator> python.exe .\combat_action_gen.py

check out.txt
